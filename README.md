# IPL Analytics Dashboard

## Project Overview
Interactive Power BI dashboard built using IPL 2022 datasets.

## Features
- Multi-page dashboard
- Team Analysis
- Player Analysis
- KPI Cards
- DAX Measures
- Dynamic Titles
- Tooltips
- Navigation Buttons
- Synchronized Slicers

## Tools Used
- Power BI
- DAX
- Power Query
- Data Modeling
